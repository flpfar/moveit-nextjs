# Moveit

> A NextJS app that gives you an exercise challenge after a 25 minutes timer.

## Built With

- Typescript
- React
- NextJS

## Setup

- Clone this repository with: `git clone https://github.com/flpfar/moveit-next.git`
- Navigate to the project folder `cd moveit-next`
- Run `yarn install`.
- Run `yarn dev`.
